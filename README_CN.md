# Rubber Duck

这是一个可以帮助你完成日常工作中不得不处理的麻烦琐事的AI小工具。

## 🦆可以成为什么？

### 你的文件夹整理助手

你可以通过**自然语言**描述你的文件需要如何整理，他会直接将目标文件夹下的内容整理成你想要的样子。

**指令:**
```txt
- 选出Movies中大小小于20MB的视频，标为待删除
- 选出md5值相同的图片，只保留一份，其余标为待删除
- 把只有图片的文件夹从Movies/中转移到Pictures/中，并按照其在Movies中的文件树保存
- 将要删除的内容存放到Temp/文件夹中，在我确认后删除
```

### 你的工作协作者

🦆希望通过聆听你对工作行为/代码内容的思考，加上自身对代码的理解，来协助你发现代码中的问题。你可以使用自然语言与他对话，他会依此做出一些反应。例如补充注释，寻找bug，发现corner case等。

**指令:**
```txt
ask:        removeHistory这个函数被我们用来移除不用的历史文件
ask:        但是他在xxxx情况下可能会有偶发的问题，导致无法移除历史文件
ask:        也许是parallelRemove这个函数导致的线程安全问题，我无法确定，你能帮我分析一下吗
---
response:   我会补充注释，帮助你分析一下parallelRemove这个函数的线程安全问题，并且看看是否有其他问题导致无法移除历史文件
```

### 你的私人秘书

你可以通过选定一系列会议纪要，让他帮忙整理成大纲还有发言稿。

**指令:**
```txt
- 当前文件夹下有近期的会议纪要，将其内容总结一下，并以 智能工作助手 为主题生成大纲和发言稿。
```

或者你可以让他做你的周报写手。

**指令:**
```txt
- 我给你的截图中记录了我本周的工作内容，帮我生成一下周报。
```
